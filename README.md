
# TaPTaP Reusable Learning Game Engine

This repository contains the blueprint for a reusable web-based learning game engine.

## Features

- JSON-driven configuration
- Modular architecture
- Reusable engine components
- Multiple games from one engine

## Components

- Core Engine Loop
- Config Loader
- Game Logic Module
- Input Handler
- Renderer
- Score Manager
